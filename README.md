#  Sistema de gestión de vuelos
#  ------------Orbitix---------------------
# Objetivo del proyecto

Este proyecto en Java e interfaz gráfica tiene como objetivo desarrollar un sistema de gestión de vuelos para la empresa VUELOS FIS que permita:

- Iniciar sección o crear una cuenta desde cero
- Filtrar el origen, destino y fecha del vuelo buscado
- Elegir el número de asientos y clase de los mismos
- Listar todos los datos del pasajero que va a ocupar cada asiento
- Seleccionar el método de pago
- Ver el historial de compras por cuenta creada en el sistema Orbitix
- Visualizar la factura y ticket de compra.

*NOTA:* El sistema incluye control de acceso mediante autenticación de usuarios con un máximo de 3 intentos por correo.

##  Instalación y Ejecución en NetBeans

Sigue estos pasos para descargar y ejecutar el proyecto **Orbitix** en tu entorno local:

### 1. Clonar y Abrir el proyecto
- Inicia **NetBeans**.
- Ve al menú superior: **Team > Git > Clone...**
- Ingresa la URL del repositorio (ej: `[(https://github.com/tu-usuario/ORBITIX_SISTEMA-2.0.git4)`] y tus credenciales si es necesario.
- Sigue los pasos del asistente y al finalizar, selecciona **Open Project**.
- NetBeans cargará automáticamente la estructura del proyecto (carpetas `src`, paquetes, dependencias, etc.).

### 2. Construir el proyecto (Build)
- Haz clic derecho sobre el nombre del proyecto (icono de la taza de café ) en el panel izquierdo.
- Selecciona **"Clean and Build"** (Limpiar y Construir) o presiona el icono del martillo con la escoba en la barra de herramientas.
- Esto descargará las librerías necesarias y verificará que no existan errores de compilación.

### 3. Ejecutar el programa
- Haz clic en el botón **"Run Project"** (icono del triángulo verde ▶️ en la barra superior) o presiona la tecla **F6**.
- Si NetBeans te pide seleccionar la clase principal, elige: `com.mycompany.orbitix.ORBITIX`.
- Se abrirá la ventana de inicio del sistema.

### 4. (Opcional) Generar ejecutable para distribuir
- Una vez finalizado el proceso de "Clean and Build" (Paso 2), ve a la carpeta del proyecto en tu explorador de archivos.
- Entra en la carpeta `dist`.
- Allí encontrarás el archivo `.jar` (ej: `Orbitix.jar`) que puedes ejecutar haciendo doble clic (requiere tener Java instalado).
